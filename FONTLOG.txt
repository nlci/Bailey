FONTLOG
Bailey - NLCI Modified Rachana Malayalam fonts
==============================================

This file provides detailed information on the NLCI Bailey fonts.
This information should be distributed along with the NLCI Bailey fonts and any derivative works.

Basic Font Information
----------------------

Malayalam font based on Rachana supporting an orthography in between the traditional orthography of Rachana
and the Malayalam reformed orthography.
